Comment l'utiliser ?

  • Pour l'utiliser vous devait :
  • 1. Créer un salon ,
  • 2. Allez dans Parametres du salon > Integrations > creer une Integrations > copiers l'url
  • 3. Allez sur le DISCORD et demandais une api KEY ,
  • 4. Une fois l'api key (=keyGF) , vous devait ajouter cette ligne de code ( dans la balises BODY )
